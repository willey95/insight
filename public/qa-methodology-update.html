01 배경 및 수정 의도
현재 도움말(PART I 비교표·PART VII 권장 조합)과 가이드 슬라이드는 방법론의 절대적 우열로 읽힐 수 있는 표현을 포함하고 있습니다. 이는 통계적 일반론으로 오해되어 사용자가 특정 방법론을 맹목적으로 선택하거나, 반대로 다른 방법론 결과를 불신하는 원인이 됩니다.
핵심 문제
P1 "LP가 유일하게 강건" — 맥락 없는 단정
LP(Local Projections)는 대형 구조충격의 horizon별 반응 추정에 유연하지만, 소표본에서는 신뢰구간이 급격히 확대됩니다. 전쟁기(19개월)나 최근 12개월 구간에서 LP를 "유일하게 강건"으로 표시하면 결과를 과신하게 됩니다.
P1 "전체 구간이 최적" — 희석 효과 미안내
전체 60개월 구간은 평균적 장기 구조 파악에 상대적으로 적합하지만, 전쟁기 극단 충격 반응이 평균에 희석됩니다. 단순히 "최적"이라 표시하면 이 한계가 전달되지 않습니다.
수정 방향
이 페이지는 "어떤 모델이 더 우수한가"를 보여주는 화면이 아니라,
분석 목적 / 표본 구간 / 충격 성격 / 표본 크기에 따라 어떤 방법을 어떻게 써야 하는가를 안내하는 화면으로 정리합니다.

05 P2 — 계산 전 Validation 로직 권장
방법론을 실행하기 전 표본 크기(n_obs)와 파라미터 수(n_params) 비율을 검사하여 사용자에게 경고를 표시합니다.
CHG-11 · 권장 검사 로직 (pseudo-code)
// 호출 위치: runSimulation() 내, 모델 추정 직전
function validateSampleSize(periodKey, method) {
    const p = ANALYSIS_PERIODS[periodKey];
    const n = p.endIdx - p.startIdx + 1;   // 표본 크기
    const k = DRIVER_KEYS.length * 4 + 1;  // ARDL 계수 수 (~41)

    if (method === 'ardl') {
        if (n <= k)
            return { level: 'block',  msg: `n(${n}) ≤ k(${k}): ARDL 추정 불가. Gaussian으로 전환 권장` };
        if (n <= k * 2)
            return { level: 'error',  msg: `n/k = ${(n/k).toFixed(1)} — 추정 매우 불안정. 탐색적 해석만 권장` };
        if (n <= k * 3)
            return { level: 'warn',   msg: `n/k = ${(n/k).toFixed(1)} — 과적합 주의. 결과 방향성 위주 해석` };
    }
    if (method === 'lp') {
        const h = 12;  // max horizon
        if (n < 30 + h)
            return { level: 'warn', msg: `LP horizon ${h}M에서 n(${n}) 부족 — 신뢰구간 넓음, 방향성 중심` };
    }
    return { level: 'ok', msg: '' };
}
경고 표시 UI 규칙
level색상동작
block빨강 (#fb7185)실행 차단 + 대체 방법론 자동 선택 안내
error빨강 반투명실행 허용, 결과 상단에 빨강 경고 배너 표시
warn노랑 (#f59e0b)실행 허용, 결과 상단에 노랑 경고 배너 표시
ok—경고 없음
전체 구간 + LP 조합 특별 경고
전체(60M) + LP 선택 시: "LP는 각 horizon에서 독립 추정합니다. 전체 구간에서는 전쟁기·안정기 구조가 혼재하므로 결과는 두 국면의 평균 반응에 가깝습니다."
06 P3 — 결과 해석 박스 / 배지 개선 선택
CHG-12 · 결과 카드 하단 자동 해석 박스
현재 방법론 + 기간 조합에 따라 해석 안내 문구를 자동으로 표시합니다.
방법론기간표시 문구
Gaussian 전체 가정 고정형 baseline입니다. 데이터 기반 추정치와 비교하여 방향성 일치 여부를 확인하세요.
ARDL 전쟁기 n=19, 계수 수=41 — 자유도 부족. 수치보다 방향성 중심으로 해석하세요.
ARDL 전체 장기 평균 구조를 반영합니다. R² 확인 후 낮으면 계수 해석 주의.
LP 전쟁기 구조충격 방향성 추정에 조건부로 유연합니다. 소표본(19M)으로 신뢰구간이 넓을 수 있습니다.
LP 최근 12M n=12, horizon 12M — n < 30+h 조건 미충족. 탐색적 참고 자료로만 활용하세요.
앙상블 전체 3가지 방법론의 단순 평균입니다. 방법론 간 편차가 클수록 불확실성이 높습니다.
