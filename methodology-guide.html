Method 1 — Gaussian IRF

Gaussian Impulse Response Function
고정 가중치 + Gaussian 형태 충격반응함수
각 거시변수에 대해 사전 지정된 가중치(weight), 최대 반응 시점(peakLag), 확산폭(width)을 사용하여 충격의 시간별 전파를 모델링합니다.

수학적 정의

$$\Delta CC_h = \sum_{k} \text{shock}_k \times w_k \times \exp\!\left(-\frac{(h - \mu_k)^2}{2\sigma_k^2}\right) \times 0.1$$

여기서 $w_k$는 드라이버 $k$의 전달 가중치, $\mu_k$는 peak lag(최대 반응 시점), $\sigma_k$는 확산폭입니다. 관성 감쇠는 별도로 적용됩니다:

$$\text{inertia}(h) = \max\!\big(0.1,\; \overline{\Delta CC}_{MoM} \cdot e^{-h/10}\big)$$

작동 원리

  1. 각 드라이버(환율, 유가 등)에 대해 고정된 3개 파라미터(weight, peakLag, width)를 사전 설정
  2. 사용자가 입력한 충격(shock %)에 가중치를 곱하고, Gaussian 커널로 시간별 반응 크기를 계산
  3. 12개월간 기준성장률 + 충격성장률을 누적하여 전망 경로를 생성

장점

  • 데이터 없이도 작동 (파라미터만 설정)
  • 소표본에서도 안정적
  • 직관적이고 설명이 쉬움
  • 계산 즉시 완료

한계

  • 가중치가 추정이 아닌 가정 — 실제와 괴리 가능
  • IRF 형태가 항상 Gaussian — 실제 반응은 비대칭일 수 있음
  • 변수 간 상호작용 미반영 (환율↑→수입물가↑ 연쇄 무시)
  • 구조 변화(전쟁, 정책 전환)에 적응 불가
적합한 사용 시점: 데이터가 부족하거나, 빠른 직관적 판단이 필요할 때. "대략적인 방향과 규모"를 파악하는 초기 스크리닝에 적합합니다.

Method 2 — ARDL-OLS

Autoregressive Distributed Lag
데이터 기반 시차 계수 추정 (OLS)
선택한 분석 기간의 실제 데이터로 각 드라이버의 시차별 영향 계수를 OLS로 추정합니다. "이 기간에 환율 1% 상승이 2개월 후 건설공사비에 실제로 얼마나 영향을 미쳤는가"를 데이터가 답합니다.

수학적 정의

$$\Delta CC_t = \alpha + \sum_{k=1}^{K} \sum_{l=0}^{L} \beta_{k,l} \cdot \Delta X_{k,t-l} + \varepsilon_t$$

여기서 $\Delta CC_t$는 건설공사비의 월별 변화율, $\Delta X_{k,t-l}$은 드라이버 $k$의 $l$개월 전 변화율입니다. 계수 $\beta_{k,l}$은 OLS로 추정됩니다:

$$\hat{\boldsymbol{\beta}} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{y}$$

$R^2$로 모델의 설명력을 검증합니다. 현재 구현에서는 $K=8$ (8개 드라이버), $L=4$ (최대 4개월 시차)로 총 $8 \times 5 + 1 = 41$개 계수를 추정합니다.

작동 원리

  1. 선택된 분석 기간의 시계열을 월별 변화율(Δ%)로 변환
  2. 각 관측치에 대해 8개 드라이버 × (lag 0~4) = 40개 설명변수 + 상수항으로 X 행렬을 구성
  3. OLS 회귀: $\mathbf{X}$ 전치 → 곱셈 → 역행렬 → 계수 벡터 $\hat{\beta}$ 산출
  4. 추정된 시차별 계수로 IRF를 구성 — Gaussian 가정 없이 데이터가 결정한 형태
  5. 충격 입력 시 추정 계수를 누적 적용하여 12개월 전망 생성

장점

  • 데이터 기반 — 주관적 파라미터 제거
  • 시차별 비대칭 반응 포착 가능
  • R² 제공 — 모델 적합도 정량 검증
  • 분석 기간 변경 시 계수가 자동 재추정
  • 변수 간 간접 상호작용을 시차 구조로 포착

한계

  • 표본 크기가 작으면 계수 추정이 불안정 (과적합 위험)
  • 41개 계수 vs ~20개 관측치 → 전쟁기 단독 추정 시 주의
  • 재귀 전망 시 오차가 누적될 수 있음
  • 구조 변화를 추정 기간 내에서만 반영
적합한 사용 시점: 충분한 관측치(30+)가 있는 기간에서 변수별 영향 크기를 정량적으로 파악하고 싶을 때. "전체 기간"이나 "최근 12개월" 설정에서 가장 안정적입니다.

Method 3 — Local Projections (Jordà, 2005)

Local Projections — Direct Impulse Response
각 시점(horizon)에서 직접 추정하는 충격반응
Oscar Jordà(2005)가 제안한 방법론으로, VAR/ARDL의 재귀적 IRF 대신 각 예측 시점 h에서 별도의 회귀를 실행합니다. 전쟁 같은 대형·비선형 충격에 가장 강건한 방법론입니다.

수학적 정의

각 horizon $h = 1, 2, \ldots, 12$에 대해 독립적인 회귀를 실행합니다:

$$y_{t+h} - y_t = \alpha_h + \sum_{k=1}^{K} \beta_{h,k} \cdot \Delta X_{k,t} + \varepsilon_{t+h}$$

여기서 좌변은 시점 $t$부터 $t+h$까지의 누적 변화이고, $\beta_{h,k}$는 "드라이버 $k$가 1단위 변화했을 때, $h$개월 후까지의 누적 영향"을 직접 추정합니다.

핵심은 12개의 별도 OLS 회귀를 실행한다는 점입니다. ARDL은 1개 회귀의 계수를 재귀적으로 누적하지만, LP는 각 시점의 영향을 독립적으로 추정합니다.

전쟁·금융위기 같은 대형 충격은 기존 선형 관계를 깨뜨립니다. VAR/ARDL은 "평상시 관계"를 가정하고 재귀 적용하므로 극단 시나리오에서 오차가 폭발합니다. LP는 각 시점에서 독립적으로 추정하므로 비선형 반응도 포착하고 오차가 누적되지 않습니다.

장점

  • 오차 비누적 — 장기 전망에서도 신뢰구간이 폭발하지 않음
  • 비선형 반응 포착 — 전쟁 충격의 비대칭적 전파 반영
  • 모델 misspecification에 강건 (VAR 차수 선택 오류 영향 없음)
  • 직관적 해석: "$\beta_{6,\text{oil}}$은 유가 1% 충격의 6개월 후 누적 영향"

한계

  • h가 커질수록 유효 표본이 줄어 추정 정밀도 하락
  • 소표본(< 20)에서 불안정 — 전쟁기 단독 사용 시 주의
  • Newey-West 표준오차 등 추가 보정이 이상적 (현재 미구현)
  • 단기(1~3M) 추정은 ARDL보다 효율성이 낮을 수 있음
적합한 사용 시점: 전쟁, 금융위기, 공급망 붕괴 등 대형 비선형 충격의 전파를 분석할 때. 특히 "전쟁기 데이터로 추정 + LP 방법론"이 가장 의미 있는 조합입니다.

구현 세부사항

OLS 행렬 연산 (순수 JavaScript)

외부 라이브러리 없이 행렬 전치, 곱셈, 역행렬(가우스-조르단 소거법)을 구현합니다:

matTranspose(A) — $m \times n \to n \times m$
matMul(A, B) — $A_{m \times p} \cdot B_{p \times n} \to C_{m \times n}$
matInvert(M) — Gauss-Jordan elimination, partial pivoting
olsEstimate(X, y) — $\hat{\beta} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{y}$, $R^2$ 계산 포함

모델 캐싱

ARDL과 LP 모델은 분석 기간별로 한 번 추정 후 _modelCache에 저장됩니다. 기간을 변경하면 캐시가 초기화되고 재추정됩니다. 충격(shock) 값 변경 시에는 재추정 없이 기존 계수를 적용합니다.

분석 결과 해석 가이드

PRESSURE BADGE 해석
HIGH PRESSURE — 복합 점수 > 3.0. 건설비 상승 압력이 강하게 작용 중.
WATCH — 복합 점수 1.5~3.0. 일부 변수에서 압력 신호 감지.
STABLE — 복합 점수 < 1.5. 현재 유의미한 상승 압력 없음.
3M / 6M / 12M 추가압력 해석
+2.5% (3M) → 향후 3개월간 건설공사비 지수가 현재 대비 2.5% 추가 상승 예상
-0.3% (6M) → 6개월 시점에서 충격이 흡수되어 소폭 하락 전환 예상
3방법 비교 시 — 결과가 동일 방향이면 확신↑, 서로 다르면 불확실성 구간에 해당
ATTRIBUTION TABLE 해석
기여도(%) — 해당 변수가 전체 공사비 변동의 몇 %를 설명하는지 (change × weight).
반영시차 — 충격 후 최대 영향까지 걸리는 개월 수 (peakLag).
전달경로 — 충격이 건설공사비까지 도달하는 중간 경로.
기여도가 가장 큰 변수 = 핵심 관리 대상. 시차가 긴 변수는 선제 대응 가능.
상관관계 vs 인과관계 주의
상관계수(r)가 높다고 인과관계를 의미하지 않습니다.
r > 0.7 — 강한 양의 상관. 함께 움직이는 경향 강함.
-0.7 < r < 0.7 — 중간~약한 관계. 다른 요인 존재 가능.
인과 방향 확인에는 Granger 인과검정 또는 구조 VAR이 필요하며, 본 대시보드는 예측적 상관에 초점.
